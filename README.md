# 🐍 OLX Pro Scraper: Python Automation
A professional-grade web scraper built to extract listings from OLX. This project showcases my skills in **Python**, **Selenium**, and **Data Automation**.

## 🚀 Features
* **Automated Browsing:** Uses Selenium to navigate OLX.
* **Smart Extraction:** Scrapes Product Titles, Prices, and Locations.
* **Data Privacy:** Built with ethical scraping practices (Rate limiting).
* **Export:** Saves data directly into Excel/CSV formats.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Library:** Selenium, Pandas
* **Driver:** Chrome WebDriver

## 💼 Business Use Case
This tool can be used for market research, price tracking, and competitor analysis.
